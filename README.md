# Vi Editor

* The soul editor of shell VI, this script will change the look of vi development env.
     
** Create a new file 
        ```css
        
        $ ~\.vimrc

        ```
** Add the following code in file
        
        ```css

        syntax on
        colorscheme pablo
        :set number
        :set tabstop=8
        :set expandtab
        :set shiftwidth=4
        :set autoindent
        :set smartindent
        :set cindent
        
        ````
